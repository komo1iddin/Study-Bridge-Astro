ImagesSection Test Page

Navigation Options

There are several ways to navigate to the Images tab:

  1. URL Hash Navigation

    Append #images to the URL:

    http://localhost:4321/en/universities/test-university#images
    Try Direct URL Navigation
  2. Tab Navigation

    Click on the "Images" tab in the navigation tabs

  3. View Gallery Button

    Click on the "View Gallery" button in the sidebar

Debugging Notes

If the ImagesSection is not displaying correctly:

  1. Open browser console to view debug messages
  2. Verify that the university object has galleryCategories data
  3. Check network requests for any errors
  4. Confirm the tab navigation is working properly

Manual Testing Script

// In browser console:
// Wait for the component to be mounted
setTimeout(() => {
  // Get the Svelte component instance
  const component = document.querySelector('astro-island[component-url*="UniversityDetailPage"]')?.__svelte?.[0];
  
  if (component) {
    // Access component state
    console.log('Component mounted:', component.mounted);
    console.log('Active tab:', component.activeTab);
    console.log('University data:', component.university);
    
    // Force change to images tab
    component.activeTab = 'images';
  } else {
    console.log('Component not found or not mounted yet');
  }
}, 2000);
